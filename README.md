Java Sales System
  A simple and educational project for sales management, built in pure Java. It allows registering products, customers, and sales, with automatic calculations for totals. Perfect for beginners in object-oriented 
  programming, showcasing concepts like classes, inheritance, and list manipulation.

Key Features:

  Product registration (name, price, quantity).
  Sales recording with items and customers.
  Total calculations and basic report generation.
  Technologies: Java (no external frameworks).
  How to Use: Clone the repository, compile, and run the main class. Contributions are welcome!

Sistema de Vendas em Java
  Um projeto simples e educacional para gestão de vendas, desenvolvido em Java puro. Permite cadastrar produtos, clientes e registrar vendas, com cálculos automáticos de totais. Ideal para iniciantes em            
  programação orientada a objetos, demonstrando conceitos como classes, herança e manipulação de listas.

Funcionalidades principais:
  Cadastro de produtos (nome, preço, quantidade).
  Registro de vendas com itens e clientes.
  Cálculo de totais e geração de relatórios básicos.
  Tecnologias: Java (sem frameworks externos).
Como usar: Clone o repositório, compile e execute a classe principal. Contribuições são bem-vindas!
